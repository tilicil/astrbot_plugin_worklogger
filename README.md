这是一个工作日志插件
主要功能：1.帮你记录每日工作内容。
         2.每天定时发送昨天的工作总结，以图片的形式发送。
         3.可以根据工作内容生成周报，月报。
         4.可以将多个会话窗口的工作记录保存在同一个工作组中，方便从任何设备记录工作。
剩余工作：
         1.在生产环境中进行测试。
